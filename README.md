# PEACH – Conversor de Temperatura

Peach es una aplicación web sencilla desarrollada con HTML, CSS y JavaScript. Permite convertir valores entre Celsius, Fahrenheit y Kelvin mediante una interfaz clara y fácil de usar. Está diseñada para ampliarse en el futuro a nuevos módulos como medición de pH u otras unidades.

## Características
- Conversión entre Celsius, Fahrenheit y Kelvin.
- Validación básica de entrada numérica.
- Interfaz responsive y minimalista.
- Código organizado para facilitar futuras ampliaciones.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Uso
1. Ingresa un valor numérico.
2. Selecciona la escala de origen.
3. Visualiza la conversión automática en las demás escalas.

## Estructura del proyecto
- `index.html`: estructura principal de la página.
- `style.css`: estilos de la interfaz.
- `script.js`: lógica de conversión.
